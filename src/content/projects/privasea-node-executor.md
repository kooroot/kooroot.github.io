## Overview
Privasea Acceleration Node의 설치 및 실행을 자동화하는 Shell 스크립트 프로젝트입니다. Privasea는 Fully Homomorphic Encryption(FHE) 기반의 프라이버시 보존 AI 네트워크로, 암호화된 데이터에서 머신러닝 연산을 수행합니다. Linux와 macOS 환경을 지원합니다.

## Key Features

### One-Click Installation
- Linux / macOS 전용 스크립트 각각 제공
- Docker 기반 Privasea Acceleration Node 자동 배포
- screen 세션 전환 후 스크립트 자동 실행

### Wallet & Key Management
- 노드 최초 설정 시 키스토어 비밀번호 자동 생성 프로세스
- Docker 로그에서 노드 주소 확인 기능
- 웹 대시보드와 지갑 연동 가이드 포함

### Dashboard Integration
- Privasea 웹 대시보드에서 노드 상태 모니터링
- 노드 주소와 지갑 연결을 통한 리워드 관리
- Docker 로그 기반 노드 구동 상태 확인

## Technologies
- **Scripting**: Shell (Bash)
- **Container**: Docker
- **Encryption**: FHE (Fully Homomorphic Encryption)
- **Network**: Privasea AI Network, Arbitrum Sepolia

## Results
Privasea 테스트넷에서 안정적인 노드 운영을 달성하고, 암호화된 AI 추론 작업에 기여했습니다.
