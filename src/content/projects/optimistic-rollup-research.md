## Overview
A research project focused on collecting and analyzing on-chain data from Optimistic Rollup networks to identify strategies for operating rollups with lower resources. Includes both raw analytics data and automated data collection tools.

## Key Research

### Data Collection Pipeline
- Built Python-based data collector using Alchemy API
- Automated on-chain data extraction from Optimistic Rollup networks
- Structured raw data storage for reproducible analysis

### Resource Optimization Analysis
- Analyzed transaction throughput and gas consumption patterns
- Identified resource bottlenecks in rollup operation
- Documented strategies for cost-efficient rollup infrastructure

## Technologies
- **Data Collection**: Python, Alchemy API
- **Analysis**: Raw data processing, statistical analysis
- **Network**: Optimism, Ethereum L2

## Results
Produced comprehensive analytics datasets and insights for operating Optimistic Rollups with optimized resource allocation, contributing to Layer 2 scalability research.
