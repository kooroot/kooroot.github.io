## Overview
A collection of DeFi protocol implementations in Solidity covering core primitives — decentralized exchange (AMM), lending/borrowing protocol, and general DeFi contract patterns. Built with Hardhat for development and testing.

## Key Features

### DEX Implementation
- Automated Market Maker (AMM) contract
- Liquidity pool management with LP tokens
- Token swap functionality with fee mechanisms

### Lending Protocol
- Collateralized lending and borrowing system
- Liquidation logic and health factor management
- Interest rate model implementation

### DeFi Patterns
- Token standards and utility contracts
- Protocol interaction patterns
- Comprehensive test suites

## Technologies
- **Smart Contracts**: Solidity
- **Framework**: Hardhat
- **Testing**: JavaScript, Hardhat test suite
- **Patterns**: AMM, Lending, ERC-20
