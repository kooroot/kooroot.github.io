## Overview
Led the specification and smart contract development for GiwaNFT, an NFT project built on ERC-721 with custom minting mechanics and metadata management.

## Key Contributions

### Specification
- Authored comprehensive technical specification document
- Defined token metadata schema and IPFS storage architecture
- Designed minting phases (allowlist, public) with price tiers
- Specified royalty implementation using ERC-2981

### Smart Contract Development
- Implemented ERC-721A for gas-optimized batch minting
- Built allowlist verification using Merkle proof validation
- Developed reveal mechanism with provenance hash for fairness
- Implemented withdrawal and fund distribution logic

### Integration
- Designed contract interface for frontend minting dApp integration
- Specified metadata API endpoints for marketplace compatibility
- Ensured OpenSea and major marketplace metadata standard compliance

## Technologies
- **Standards**: ERC-721A, ERC-2981, ERC-165
- **Smart Contracts**: Solidity, Foundry
- **Storage**: IPFS, metadata API

## Impact
Delivered a complete NFT project from specification through smart contract deployment, covering the full lifecycle from design to marketplace listing.
