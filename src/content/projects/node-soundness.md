## Overview
A one-touch installation script for the Soundness Layer CLI, enabling quick participation in the Soundness Layer testnet. Auto-detects OS and shell environment, installs dependencies, and generates key pairs for testnet registration.

## Key Features

### Automated Setup
- OS auto-detection (Ubuntu / macOS)
- Shell detection (bash / zsh) for proper environment configuration
- Dependency installation and Soundness CLI setup

### Key Management
- Automatic key pair generation for testnet
- Testnet registration instructions and guidance
- Secure local key storage

## Technologies
- **Scripting**: Shell (Bash)
- **Infrastructure**: Linux (Ubuntu), macOS
- **Network**: Soundness Layer Testnet
