## Overview
A full-stack application that connects to the Kalshi API to analyze a user's trading history, determine their trading personality type, and provide smart market recommendations based on their preferences. Generates shareable trader profiles.

## Key Features

### Trading Analysis
- Trading history analysis via Kalshi API
- Trading personality type classification
- Personalized market recommendations

### Shareable Profiles
- Auto-generated trader profile cards
- Shareable profile links

### Architecture
- Hono backend with SQLite storage
- React frontend with TanStack Router/Query
- shadcn/ui component library

## Technologies
- **Backend**: Hono, Bun, SQLite
- **Frontend**: React, TanStack Router/Query, shadcn/ui
- **API**: Kalshi Demo API
