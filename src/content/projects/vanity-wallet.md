## Overview
A Python-based vanity wallet address generator that creates cryptocurrency wallet addresses matching specific user-defined patterns. Generates addresses with custom prefixes or suffixes for personalized wallet identification.

## Key Features

### Address Generation
- Custom prefix and suffix pattern matching
- High-performance key pair generation
- Ethereum-compatible address output

### Security
- Local key generation with no external dependencies
- Private key handling with secure output

## Technologies
- **Language**: Python
- **Cryptography**: ECDSA key pair generation
- **Network**: Ethereum address format
