## Overview
A decentralized counter application built using Arbitrum Stylus, where the smart contract is written in Rust instead of Solidity. Demonstrates the capability of deploying Rust-based contracts on Arbitrum with a React frontend for user interaction.

## Key Features

### Rust Smart Contract
- Counter contract written in Rust using Stylus framework
- Deployed via cargo-stylus toolchain
- Testnet deployment with public verification

### Frontend Interface
- React-based web application for counter interaction
- Web3 wallet connection for transaction signing
- Real-time contract state display

## Technologies
- **Smart Contract**: Rust, Arbitrum Stylus, cargo-stylus
- **Frontend**: React, TypeScript
- **Tools**: Foundry, Docker
- **Network**: Arbitrum Testnet
